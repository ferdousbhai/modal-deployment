# Modal Deployment Plugin

Run Python code in the cloud with serverless containers, GPUs, and autoscaling using [Modal](https://modal.com).

## When to Use

- Deploying ML models with GPU acceleration
- Running batch jobs and parallel data processing
- Scheduling cron jobs in the cloud
- Serving APIs with serverless infrastructure
- LLM inference and model training/fine-tuning

## Features

- Zero-config serverless Python execution
- GPU support (A10G, A100, H100)
- Automatic scaling and cold start optimization
- Built-in storage (volumes, dicts, queues)
- Web endpoints and scheduled functions

## Installation

```bash
/plugin install modal-deployment@cloud-fullstack-skills
```
